3. Professional Excel & Data Analysis

Duration
6 Days
Training Hours
30–36 Hours

6-Day Module Breakdown

Day 1 — Advanced Formulas & Lookup Functions
  ✦  Lookup & Reference Functions
› VLOOKUP: syntax, exact vs. approximate match & common errors (col_index pitfall)
› HLOOKUP: row-based lookup for horizontal datasets like rate cards
› INDEX-MATCH: why it beats VLOOKUP – left-side lookup & dynamic column reference
› XLOOKUP (Office 365): multi-column return, wildcard match & search from last
  ✦  Logical & Conditional Functions
› Nested IF: building grade bands (A/B/C/D/F) from raw score data
› IFS function: cleaner alternative to deeply nested IF statements
› IFERROR: wrapping VLOOKUP to return 'Not Found' instead of #N/A
› AND, OR, NOT: combining multiple conditions in IF & data validation formulas

Day 2 — Text, Date & Statistical Functions
  ✦  Text & String Functions
› CONCATENATE / & operator / TEXTJOIN for combining name + department fields
› LEFT, RIGHT, MID, LEN: extracting portions of text (ID codes, phone suffixes)
› TRIM, CLEAN, PROPER, UPPER, LOWER: fixing messy imported text data
› FIND, SEARCH, SUBSTITUTE, REPLACE: locating & swapping text within strings
  ✦  Date & Statistical Functions
› TODAY(), NOW(), DATE(), YEAR(), MONTH(), DAY() for date calculations
› DATEDIF: calculating age (years) or tenure (months) from birthdate/join date
› EDATE: project due dates; EOMONTH: last day of month for billing cycles
› COUNTIF, SUMIF, AVERAGEIF with wildcard (*) and multiple criteria versions

Day 3 — Pivot Tables & Dynamic Reporting
  ✦  Pivot Tables – Build & Customize
› Creating a Pivot Table from a 500+ row sales dataset
› Row/column fields: product category in rows, region in columns, revenue in values
› Grouping dates by month/quarter/year in Pivot Table automatically
› Calculated field: profit margin = (revenue−cost)/revenue directly in Pivot Table
  ✦  Pivot Charts & Slicers
› Inserting a Pivot Chart: bar, line & combo chart types linked to Pivot Table
› Chart formatting: axis titles, data labels, gridlines & color palette customization
› Slicers: adding region & category slicers and linking to multiple Pivot Tables
› Timelines: filtering Pivot Table by month/quarter with a date timeline slicer

Day 4 — Dashboard Design & Data Validation
  ✦  Excel Dashboard Design
› Dashboard layout: dedicated sheet with linked charts, KPI cells & slicers
› Named ranges: using meaningful names in formulas (=SUM(Revenue) vs =SUM(D2:D100))
› Dynamic chart titles: linking title cell to a slicer-selected value via formula
› Print-ready formatting: removing gridlines, adding logo & setting print area
  ✦  Data Validation & Protection
› Dropdown lists from a range: product names, department list, status (Active/Inactive)
› Custom validation: accept only values between 1–100 or dates in current year
› Error alerts: Stop (block invalid entry), Warning & Information message types
› Sheet & workbook protection: locking formula cells while leaving input cells editable

Day 5 — Power Query & Data Cleaning
  ✦  Power Query – Import & Transform
› Connecting to CSV, Excel & folder of files as a single query
› Removing duplicate rows, filtering blanks & replacing error values in Power Query
› Splitting columns: full name → first/last; date → day/month/year
› Merging two tables (JOIN): combining sales table with product master by product ID
  ✦  Data Cleaning Best Practices
› TRIM & CLEAN functions vs. Power Query 'Trim' transform – when to use each
› Standardizing inconsistent entries: 'Male'/'male'/'M' → 'Male' with PROPER or Replace
› Removing duplicates with Excel's built-in tool vs. Power Query method
› Flash Fill: auto-detecting patterns for email extraction, code formatting & name parsing

Day 6 — Capstone Project, Assessment & Certificate
  ✦  Capstone – Business Analytics Report
› Import a provided 300-row raw sales dataset using Power Query & clean it
› Build 3 Pivot Tables: by product, by region & by month with calculated margin field
› Dashboard sheet: 3 charts, 2 slicers, KPI cells (total revenue, avg margin, top product)
› Export a print-ready PDF report & present insights in 3 minutes to trainer
  ✦  Assessment & Career Guidance
› Timed test: 10 formula tasks + 1 Pivot Table + 1 dashboard in 45 minutes
› MCQ quiz: 15 questions on functions, Pivot Tables & data cleaning
› Career paths: MIS executive, data analyst, reporting analyst, business analyst

  ✦  Connecting to Data Sources
› Get Data from Excel: selecting specific sheets & tables; previewing before loading
› Get Data from CSV/Text: delimiter detection, header row & column type assignment
› Get Data from Web: pasting a URL to import public data (Wikipedia tables, govt datasets)

Day 2 — Power Query Data Cleaning
  ✦  Power Query Editor Basics
› Opening Power Query Editor: Home → Transform Data
› Applied Steps pane: every transformation recorded as a step (undo/redo any step)
› Renaming queries, removing columns & reordering columns for clean model
› Filtering rows: removing blanks, nulls & test/dummy data rows before loading
  ✦  Data Transformations
› Changing data types: text → whole number, text → date, decimal number corrections
› Splitting columns: first name/last name split; date → separate Day, Month, Year columns
› Replacing values: 'M' → 'Male', 'F' → 'Female', fixing inconsistent category names
› Merging queries: combining a sales table with a product master table by product ID (LEFT JOIN)

Day 3 — Data Modeling & Relationships
  ✦  Building a Data Model
› Model view: dragging tables & creating relationships by connecting matching columns
› One-to-many relationship: one product in Product table → many rows in Sales table
› Cardinality & cross-filter direction: single vs. both – when each is appropriate
› Star schema: fact table (Sales) surrounded by dimension tables (Date, Product, Customer, Region)
  ✦  Date Table & Time Intelligence
› Why a dedicated Date table is needed: covering all dates including those with no sales
› Creating a Date table using DAX: CALENDAR(DATE(2023,1,1), DATE(2024,12,31))
› Adding Year, Month, Month Name, Quarter & Week Number columns to Date table
› Marking the Date table: right-click → Mark as Date Table for time-intelligence DAX to work

Day 4 — DAX Measures & Core Visuals
  ✦  DAX Fundamentals
› Measures vs. calculated columns: measures are dynamic (recalculate per filter), columns are static
› SUM, COUNT, COUNTROWS, AVERAGE & DISTINCTCOUNT – the essential 5 aggregation measures
› CALCULATE: overriding filter context – e.g., CALCULATE(SUM(Sales[Revenue]), Region='North')
› DIVIDE(numerator, denominator, alternate): safe division avoiding divide-by-zero errors
  ✦  Core Report Visuals
› Bar & column charts: clustered vs. stacked; X-axis = category, Y-axis = measure
› Line chart: trend over time; combo chart: bar (revenue) + line (profit %) on dual axis
› Card visual: single KPI number (Total Revenue: ₹12.4L); Multi-row card: 4 KPIs in one visual
› Table & matrix: matrix with rows (product) × columns (region) × values (revenue) for cross-tab

Day 5 — Dashboard Design & Publishing
  ✦  Interactive Dashboard Design
› Report canvas sizing: 16:9 standard (1280×720) for screen presentations
› Slicers: dropdown, list & between (date range) – controlling all visuals on the page
› Bookmark + button navigation: creating tab-style navigation between report pages
› Conditional formatting: color scale on matrix values & icon sets (up/down arrows on % change)
  ✦  Publishing to Power BI Service
› Publishing: Home → Publish → selecting workspace in Power BI Service
› Power BI Service dashboard: pinning visuals from multiple reports onto one dashboard
› Sharing a report: 'Share' button → enter email or 'Publish to Web' for public embed link
› Row-Level Security intro: creating roles to restrict which data each user can see
